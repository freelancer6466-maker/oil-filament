Express Entry

Your Fast Track to Canadian Permanent Residence

What is Express Entry?

Express Entry is Canada's flagship immigration system that manages skilled worker applications. It's designed to select immigrants based on their ability to settle in Canada and contribute to the Canadian economy.

The system manages applications for three federal economic immigration programs:

  • Federal Skilled Worker Program
  • Federal Skilled Trades Program
  • Canadian Experience Class

Key Requirements

  • Create an online Express Entry profile
  • Meet minimum language requirements (English/French)
  • Have eligible work experience
  • Obtain an Educational Credential Assessment (ECA)
  • Prove you have enough funds to settle in Canada
  • Score enough points in the Comprehensive Ranking System (CRS)

The Process

  1. Create an Express Entry profile
  2. Get ranked in the Express Entry pool
  3. Receive an Invitation to Apply (ITA) if selected
  4. Submit a complete application within 60 days
  5. Receive a decision on your application
